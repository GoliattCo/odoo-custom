Colombian Bank Reconciliation

Odoo 18

Import bank statement flat files (CSV/TXT) from major Colombian banks and reconcile them with your accounting entries.

Supported Banks

  • Bancolombia - CSV export with semicolon delimiter
  • Davivienda - CSV export with semicolon delimiter
  • Generic CSV/TXT - Fully configurable column mapping

Features

  • Import wizard with encoding auto-detection (UTF-8, Latin-1, CP1252)
  • Auto-reconciliation by reference and amount matching
  • Manual reconciliation for unmatched lines
  • Configurable bank formats: delimiter, column mapping, date/number formats
  • Colombian number format support (dot as thousands, comma as decimal)
  • Progress tracking per imported statement

Configuration

Go to Accounting > Bank Reconciliation > Configuration > Bank Formats to create or modify bank file format definitions.
